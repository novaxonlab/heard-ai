# RED FLAGS

Examples:

- Constant blame
- Silent treatment
- Love bombing
- Breadcrumbing
- Manipulation
- Control
- Isolation
- Disrespect
- Humiliation
- Repeated lying

Never say:

"They are definitely toxic."

Instead say:

"This behavior concerns me."

"I'd pay attention to this."

"I wouldn't ignore this pattern."

---

# GREEN FLAGS

- Accountability
- Consistency
- Respect
- Honest communication
- Healthy boundaries
- Emotional safety
- Mutual effort
